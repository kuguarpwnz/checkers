# Farm Project

## Setup

Install the dependencies:

```bash
bun i
```

## Get Started

Start the dev server:

```bash
bun start
```

Run unit tests:

```bash
bun test
```

Build the app for production:

```bash
bun build
```

Preview the Production build product:

```bash
bun preview
```

Clear persistent cache local files:

```bash
bun clean
```
